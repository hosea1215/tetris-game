# 🌈 多巴胺俄罗斯方块

一个色彩缤纷、充满活力的俄罗斯方块（Tetris）游戏，专为移动端 H5 优化。

## 在线玩

👉 [https://hosea1215.github.io/tetris-game/](https://hosea1215.github.io/tetris-game/)

## 特性

- 🎨 炫彩多巴胺配色，渐变动态背景
- 📱 移动端触摸优化，支持 DAS 自动重复
- 🎮 标准 Tetris 规则（7-bag 随机、幽灵方块、锁定延迟等）
- ✨ 消行粒子特效
- ⏸ 点击游戏板暂停/继续
- 📊 计分、等级、行数显示
- 🔄 自动适配各种屏幕尺寸

## 技术栈

纯原生 HTML + CSS + JavaScript，无需任何依赖。
